## 🔧 Customization

### Update Personal Information

Edit `index.html` to change:

- Name, email, and descriptions
- Skills and proficiency areas
- Project details and descriptions
- Social media links

### Modify Colors

Edit `styles.css` and change these CSS variables:

```css
:root {
  --primary-color: #6366f1; /* Main accent color */
  --secondary-color: #ec4899; /* Secondary accent */
  --accent-color: #f59e0b; /* Additional accent */
  --dark-bg: #0f172a; /* Dark background */
  --light-bg: #f8fafc; /* Light background */
}
```

### Add More Projects

Add a new project card in the Projects section:

```html
<div class="project-card">
  <div class="project-header">
    <i class="fas fa-code"></i>
  </div>
  <h3>Your Project Name</h3>
  <p>Project description here...</p>
  <div class="project-tags">
    <span class="tag">Technology</span>
  </div>
</div>
```

### Add GitHub Links to Projects

Update project links in the HTML to point to actual GitHub repositories

## 🌐 Deployment Options

### Option 1: GitHub Pages

1. Create a GitHub repository named `portfolio`
2. Upload all files
3. Go to Settings → Pages → Select `main` branch
4. Your site will be live at `yourusername.github.io/portfolio`

### Option 2: Netlify

1. Drag and drop the portfolio folder to Netlify
2. Get a live URL instantly

### Option 3: Vercel

1. Connect your GitHub repository to Vercel
2. Deploy with one click
